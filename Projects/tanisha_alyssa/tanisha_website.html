About the prototype

  • Upon leaving home

    Sensor activates when user leaves the door

  • Every hour

    Every 1hr, relative of the user will be notified where they are and able to give them a call

  • If lost what happenes

    if they are lost the relative can activate a red flashing circle on the microbit to indicate that dementia user needs help. If anyone sees the flashing red circle, they can refer to the card on what it means and assist the user

  • If lost earlier than 1hr

    emergency red flashing circle can be activated by user from emergency stressball and deactivated by relative

  • If they are ok

    if relative calls and user is ok, green light remains

  • Able to reach home safely !
